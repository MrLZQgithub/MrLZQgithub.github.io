Python-opencv-视觉巡线

 · 2019-8-26 · 次阅读

一、概述

通过摄像头采集图像，
将图像灰度化、二值化、膨胀、腐蚀操作后，
提取第400行像素值v,接近于图像底线位置，
提取中间值（这里为白色）的数量和位置，
根据数量和位置，利用简单的数学公式，（首项+尾项）/2，计算出白色的中间位置，
然后对比实际的中间位置320（不需要改），计算出偏移量，
最后根据偏移量计算出电机应有的转角。
