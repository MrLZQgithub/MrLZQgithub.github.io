人工智能与python

应用简单了解

手机，电脑，无人机，无人驾驶，阿尔法狗等等。
人工智能->机器学习->深度学习。
python是必备工具手段。

python的优势

可以进行多平台运行，比如windows，linux，macos。。。，最好在linux系统上练习，高点有意思的事情可以用树莓派。

语法简单，容易理解，简单易学。

现在存在的强大的数据处理库，numpy（提供了许多python进行n维数组和矩阵操作的功能），scipy（提供高效的数学运算功能），pandas（数据观察），matplotlib（做熟据的可视化图表）

可以进行web开发，爬虫，图形界面开发，科学家计算框架，机器学习，操作数据库等等

python的编程利器， jupyther notebook，前身是Ipython，推推i推推，大家都说好。

就业方向

网站开发，人工智能，数据分析，linux运维，爬虫工程师，自动化测试。
